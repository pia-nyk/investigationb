# Investigation Bureau
Online Police Station Management Software 

<b>Functionalities:</b>
<ul>
  <li>Add case </li>
  <li>Assign case </li>
  <li>Add Suspects to a case </li>
  <li>Add evidences to the case </li>
  <li>Add FIR</li>
  <li>View Case history</li>
</ul>

<b>Softwares Used:</b>
<ul>
  <li>Netbeans for Java</li>
  <li>MySQL database</li>
</ul>
